# Analyse the best app

This is a simple example how to find the best N apps based on revenue.

### Requirements to use the cookiecutter template:
-----------
 - Python 3.6
-----------

``` bash
$ pip setup.py install
```

### To start , run:

Prepare data
``` bash
$ make prepare
```

Visualize
``` bash
$ make visualize
```

Show apps that need to be in the short window
``` bash
$ make visualize
```
